Export weekly small group reports into a formatted document ready for sharing or printing.

Read all reports in src/ for the most recent week and produce a single well-formatted output that includes:
- Header with church name, date, and reporting period
- Individual group summaries (one per group)
- Combined attendance totals
- Consolidated prayer requests section
- Follow-up action items

Ask the user whether they want plain text or markdown output, then write the file to src/exports/ named: YYYY-MM-DD_weekly-report-export.md
