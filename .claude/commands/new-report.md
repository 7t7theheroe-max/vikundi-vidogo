Scaffold a new weekly small group report form for the current week.

Ask the user for:
- Group leader name
- Group name
- Date of meeting

Then generate a structured report template in src/ with the following sections:
- Attendance (number present, names if available)
- Topic/Scripture covered
- Key discussion points
- Prayer requests
- Follow-up needs
- Leader notes

Name the file using the format: YYYY-MM-DD_[group-name]-report.md
